DevSecOps Tools Dir

Snort

Snort is a very powerful, open-source DevSecOps tool that helps identify intruders. It also exposes malicious attacks on the system. Allows real-time traffic analysis and package logs.

Git – Version Control Tool

One of the fundamental building blocks of any CI setup is to have a strong version control system.

Jenkins

Jenkins is an automation server. It is an open-source tool and written in Java. It helps many projects in automating, building and deploying.
